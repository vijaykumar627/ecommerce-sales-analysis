# 🛒 E-Commerce Sales Analytics

## 📌 Project Overview

This project analyzes e-commerce sales data to identify business trends, improve customer retention, optimize product performance, and support data-driven business decisions.

The project follows a complete Data Analytics workflow using Excel, SQL, Python, Power BI, and AI-generated business insights.

---

## 🎯 Business Problem

Businesses often collect large amounts of sales data but struggle to answer questions such as:

- Which products generate the highest revenue?
- Which customers are most valuable?
- Which regions perform best?
- How do discounts impact profit?
- How can customer retention be improved?

This project aims to answer these questions through data analysis.
